# Lightning Opener

Create new panes or windows super fast without pain (pun intended)

## Requirement

- [tpm](https://github.com/tmux-plugins/tpm)

## Installation

Add the following to `$HOME/.tmux.conf`

```
set -g @plugin 'hitochan777/tmux-lightning-opener'
```
