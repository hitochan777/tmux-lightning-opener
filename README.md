# Lightning Opener

Lets you open new panels lightening fast regardless of the loading time of your shell.

## Requirement

- [tpm](https://github.com/tmux-plugins/tpm)

## Install

Add the following to `$HOME/.tmux.conf`

```
set -g @plugin 'hitochan777/lightning-opener'
```
